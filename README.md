# Sistema de RH – CRUD

Este projeto consiste em um CRUD de um Sistema de Recursos Humanos, desenvolvido para a disciplina de Linguagem de Programação integrada com Banco de Dados.

O sistema foi desenvolvido utilizando Python, Flask e MySQL, e tem como objetivo realizar o gerenciamento básico de dados de RH por meio de três tabelas do banco de dados: cargo, departamento e funcionário.

Apesar de ser um projeto simples, ele foi feito com dedicação e serviu para aplicar na prática conceitos fundamentais como CRUD, relacionamento entre tabelas e integração entre aplicação web e banco de dados.
Futuramente, o sistema pode receber melhorias, principalmente no front-end e em funcionalidades adicionais.
